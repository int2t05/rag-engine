## 每步产出文件清单（简要）

### Step 1: 项目初始化
- `backend/` 目录、`requirements.txt`、`main.py` 骨架
- `frontend/` 目录、`package.json`、`next.config.js`
- `.env.example`、`.gitignore`

### Step 2: 基础设施
- `docker-compose.yml`、`docker-compose.dev.yml`
- `nginx.conf`、`backend/Dockerfile`、`frontend/Dockerfile`

### Step 3: 用户认证
- `models/user.py`、`schemas/user.py`、`schemas/token.py`
- `core/config.py`、`core/security.py`、`db/session.py`
- `api/api_v1/auth.py`、Alembic 迁移

### Step 4: 前端登录注册
- `layout.tsx`、`page.tsx`、`login/page.tsx`、`register/page.tsx`
- `lib/api.ts`、`lib/utils.ts`

### Step 5: 知识库 CRUD
- `models/knowledge.py`（KnowledgeBase、Document 基础）
- `api/api_v1/knowledge_base.py`（创建、列表、详情、更新、删除）
- `dashboard/knowledge/page.tsx`、`dashboard/knowledge/new/page.tsx`

### Step 6: 文档上传与存储
- `models/knowledge.py`（DocumentUpload、ProcessingTask）
- `core/minio.py`、文档上传 API
- `document-upload-steps.tsx`（上传步骤组件）

### Step 7: 文档解析与向量化
- `services/embedding/embedding_factory.py`
- `services/vector_store/`（base、chroma、factory）
- `services/document_processor.py`（解析、分块、向量化）
- `models/knowledge.py`（DocumentChunk）

### Step 8: RAG 对话
- `models/chat.py`、`schemas/chat.py`
- `services/llm/llm_factory.py`
- `services/chat_service.py`、`api/api_v1/chat.py`

### Step 9: 前端聊天与引用
- `dashboard/chat/page.tsx`、`chat/new/page.tsx`、`chat/[id]/page.tsx`
- `components/chat/answer.tsx`、`components/layout/dashboard-layout.tsx`

### Step 10: 增强与部署
- `models/api_key.py`、`api/api_v1/api_keys.py`
- `api/openapi/knowledge.py`
- `docker-compose` 完善、`docs/troubleshooting.md`

## 使用方式

1. 按顺序阅读每一步文档
2. 每步文档会列出：**要实现的文件**、**关键代码**、**验收标准**
3. 完成后可运行对应功能，再进入下一步
4. 遇到问题可参考 `docs/troubleshooting.md` 和 `ARCHITECTURE.md`
