# 项目总览

## 项目简介

RAG Engine 是一个基于 FastAPI 构建的 **RAG（检索增强生成）引擎** 后端服务，提供完整的知识库管理、智能对话和 RAG 评估功能。

## 核心能力

| 能力 | 说明 |
|------|------|
| **知识库管理** | 多知识库创建、文档上传/解析/分块/向量化存储 |
| **智能对话** | 基于知识库的流式 RAG 对话，多知识库关联；可配置检索流水线（混合检索、重排、引用详情页等） |
| **RAG 评估** | 基于 RAGAS 指标体系的自动化评估 |
| **灵活配置** | 支持 OpenAI / Ollama 多模型配置管理 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | FastAPI + SQLAlchemy + Alembic |
| 关系数据库 | MySQL |
| 向量数据库 | ChromaDB |
| 对象存储 | MinIO |
| AI 集成 | OpenAI / Ollama (LLM + Embeddings) |
| RAG 框架 | LangChain (LCEL) |
| 评估框架 | RAGAS (Strict JSON) |

## 边界说明

- **后端**：Python 3.11，FastAPI + SQLAlchemy + Alembic + MySQL；对象存储 MinIO；向量库以 Chroma 为主。
- **前端**：Next.js 14 + React 18 + TypeScript + Tailwind（具体页面以仓库内代码为准）。
- **模型**：不绑定单一云厂商；通过用户级配置支持 OpenAI 兼容 API 与 Ollama 等（见 `LLMFactory` / `EmbeddingsFactory`）。
