# 数据模型

## ER 图

```
┌─────────────┐       ┌─────────────────────┐       ┌─────────────┐
│    User     │───────│ LlmEmbeddingConfig  │       │             │
└─────────────┘       └─────────────────────┘       │
     │                                                  │
     │ 1:N                                           │ 1:N
     ▼                                                │
┌─────────────┐       ┌─────────────────────┐       ┌─────────────┐
│KnowledgeBase│───────│  Document           │───────│DocumentChunk│
└─────────────┘       └─────────────────────┘       └─────────────┘
     │                        │
     │ N:M              1:N   │
     ▼                        ▼
┌─────────────┐       ┌─────────────────────┐
│    Chat     │───────│  ProcessingTask     │
└─────────────┘       └─────────────────────┘
     │
     │ 1:N
     ▼
┌─────────────┐
│   Message   │
└─────────────┘
```

## 实体详解

### User

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| username | str | 用户名 |
| email | str | 邮箱 |
| hashed_password | str | bcrypt 哈希密码 |
| active_llm_embedding_config_id | int | 当前激活的 LLM 配置 |

### KnowledgeBase

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | str | 知识库名称 |
| description | str | 描述 |
| user_id | int | 所有者 FK |

### Document

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| knowledge_base_id | int | 所属知识库 FK |
| file_name | str | 原始文件名 |
| file_hash | str | SHA256 哈希（去重用） |
| minio_object_name | str | MinIO 存储路径 |

### DocumentChunk

| 字段 | 类型 | 说明 |
|------|------|------|
| id | str | SHA256 hash 作为主键 |
| document_id | int | 所属文档 FK |
| chunk_index | int | 块序号 |
| content | str | 文本内容 |
| metadata | dict | 元数据 |

### Chat

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| user_id | int | 所有者 FK |
| name | str | 对话名称 |

### Message

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| chat_id | int | 所属对话 FK |
| role | str | user / assistant |
| content | str | 消息内容（含引用上下文） |

### LlmEmbeddingConfig

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| user_id | int | 所有者 FK |
| name | str | 配置名称 |
| config_json | dict | AiRuntimeSettings JSON |

### EvaluationTask

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 主键 |
| name | str | 任务名称 |
| evaluation_type | str | full / retrieval / generation |

## 核心表清单

| 表名 | 说明 |
|------|------|
| `users` | 用户账户 |
| `knowledge_bases` | 知识库元数据（含 `parent_child_chunking`：是否父子分块入库） |
| `documents` | 已处理文档 |
| `document_uploads` | 上传临时记录 |
| `document_chunks` | 分块元数据 |
| `processing_tasks` | 文档处理任务状态 |
| `chats` | 对话 |
| `messages` | 消息 |
| `chat_knowledge_bases` | 对话↔知识库多对多 |
| `llm_embedding_configs` | 用户模型配置 |
| `evaluation_tasks` | 评估任务 |
| `evaluation_test_cases` | 测试用例 |
| `evaluation_results` | 评估结果 |
