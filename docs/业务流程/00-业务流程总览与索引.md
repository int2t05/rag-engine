## 核心流程概览

### 1. 用户注册 → 设置 LLM 配置

```
注册 → 设置 LLM/Embedding 配置 → 激活配置
```

### 2. 知识库构建

```
创建知识库 → 上传文档 → 后台处理 → Chroma 向量存储
```

### 3. RAG 对话

```
创建对话 → 关联知识库 → 发送消息（可选 rag_options）→ 模块化检索 → 生成 → SSE（引用 + 正文）
```

### 4. RAG 评估

```
创建评估任务 → 添加测试用例 → 触发评估 → 检索/生成/评分 → 汇总结果
```

## 详细模块文档

### [认证模块](./01-认证模块.md)

用户注册、登录和 JWT Token 管理，采用 OAuth2 密码模式。

### [知识库模块](./02-知识库模块.md)

文档上传、解析、分块、向量化存储和检索测试。

### [对话模块](./03-对话模块.md)

基于知识库的流式 RAG 对话，支持多知识库关联、可配置检索流水线（默认 Native 向量检索；可选查询重写、多库、混合、多路、重排、父子块等），详见该文档。

### [评估模块](./04-评估模块.md)

基于 RAGAS 指标体系的 RAG 系统自动化评估。

### [LLM 配置模块](./05-LLM配置模块.md)

多份 LLM/Embedding 配置管理，支持 OpenAI / Ollama。
