## 模块概述

评估模块提供基于 RAGAS 指标体系的 RAG 系统自动化评估能力。

**源码位置**: `app/modules/evaluation/`

| 方法 | 端点 | 功能 |
|------|------|------|
| GET | `/api/evaluation/types` | 可用评估类型/指标 |

| GET | `/api/evaluation/resolve/{task_id}` | 解析任务（如前端路由） |

```
┌─────────────────────────────────────────────────────────────────┐
│  1. 任务创建                                                       │
│     创建 EvaluationTask + EvaluationTestCase（问题+参考答案）     │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  2. 触发评估 (POST /{task_id}/run)                               │
│     background_tasks.add_task(run_evaluation_task)               │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  3. 逐个处理测试用例                                               │
│                                                                  │
│  ┌────────────────────────────────────────────────────────────┐  │
│  │ 3.1 检索 → retrieve_for_context（与对话 Native 一致的稠密向量路径）│  │
│  │ 3.2 生成 → LLM (temperature=0) 基于上下文生成答案          │  │
│  │ 3.3 评分 → RAGAS Strict JSON 指标                         │  │
│  └────────────────────────────────────────────────────────────┘  │
│                              │                                   │
│                              ▼                                   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  4. 得分计算                                                       │
│     ragas_score = (context_relevance + faithfulness +          │
│                    answer_relevance) / 3                        │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  5. 通过判定                                                       │
│     pass_rate = 各指标 ≥ 0.6 的比例                               │
└─────────────────────────────────────────────────────────────────┘
```

## RAGAS 指标

### 指标类型

| 指标 | 说明 | 范围 |
|------|------|------|
| **faithfulness** | 答案对上下文的忠诚度 | [0, 1] |
| **answer_relevance** | 答案与问题的相关性 | [0, 1] |
| **context_relevance** | 上下文的整体相关性 | [0, 1] |
| **context_precision** | 上下文的精确度 | [0, 1] |
| **context_recall** | 上下文召回率 | [0, 1] |
| **answer_correctness** | 答案正确性（需参考答案） | [0, 1] |

### 评估类型

| 类型 | 检索 | 生成 | 指标 |
|------|------|------|------|
| `full` | Yes | Yes | 全部 6 个指标 |
| `retrieval` | Yes | No | context_relevance, context_precision, context_recall |
| `generation` | Yes | Yes | faithfulness, answer_relevance |

### Strict JSON 提示词

RAGAS 评估强制输出纯 JSON，避免解析失败：

```
Given the question and context, respond with a STRICT JSON object:
{
  "faithfulness": float,
  "answer_relevance": float
}
Do not include any other text.
```

## 关键代码

### evaluation_service.py

```python
async def run_evaluation_task(task_id: int, db: Session):
    """后台评估任务执行"""
    task = db.query(EvaluationTask).get(task_id)
    test_cases = task.test_cases

    for test_case in test_cases:
        # 检索（evaluation_service._retrieve → chat.rag.retrieval_core）
        retrieved_docs = _retrieve(
            test_case.query, vector_store, top_k, db, task.knowledge_base_id
        )

        # 生成
        generated_answer = _generate_answer(
            test_case.question,
            retrieved_docs,
            llm
        )

        # 评分
        scores = _evaluate_with_ragas(
            question=test_case.question,
            answer=generated_answer,
            context=retrieved_docs,
            ground_truth=test_case.ground_truth,
            eval_type=task.evaluation_type
        )

        # 保存结果
        result = EvaluationResult(
            test_case_id=test_case.id,
            **scores
        )
        db.add(result)

    db.commit()
```

### ragas_eval.py

```python
def _evaluate_with_ragas(
    question: str,
    answer: str,
    context: list[str],
    ground_truth: str,
    eval_type: str
) -> dict:
    """RAGAS Strict JSON 评分"""

    if eval_type in ("full", "generation"):
        # faithfulness + answer_relevance
        prompt = build_ragas_prompt(question, answer, context)
        response = llm.invoke(prompt)
        scores = parse_strict_json(response)  # 严格 JSON 解析

    if eval_type in ("full", "retrieval"):
        # context_relevance + context_precision + context_recall
        scores.update(compute_retrieval_scores(question, context, ground_truth))

    return scores
```
