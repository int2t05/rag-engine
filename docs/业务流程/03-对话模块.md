## 模块概述

对话模块提供基于知识库的流式 RAG 对话能力，支持多知识库关联和历史感知检索。

**源码位置**: `app/modules/chat/`

## API 端点

### 对话 CRUD

| 方法 | 端点 | 功能 |
|------|------|------|
| POST | `/api/chat/` | 创建对话 |
| GET | `/api/chat/` | 对话列表 |
| GET | `/api/chat/{chat_id}` | 对话详情 |
| DELETE | `/api/chat/{chat_id}` | 删除对话 |

### 消息与 RAG

| 方法 | 端点 | 功能 |

| GET | `/api/chat/{chat_id}/messages` | 消息历史 |

## RAG 流程详解

```
┌─────────────────────────────────────────────────────────────────┐
│  1. 消息保存                                                       │
│     保存用户消息 + AI 消息占位到 MySQL                             │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  2. 知识库加载                                                     │
│     加载对话关联的知识库，创建 Chroma 检索器                        │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  3. History-aware Retrieval                                      │
│     用 LLM 根据对话历史重写搜索问题（LCEL）                        │
│     "之前问你关于 X 的问题，现在我想了解 Y"                         │
│     → 重写为针对 Y 的独立搜索查询                                  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  4. 文档检索                                                       │
│     Chroma similarity_search 检索相关片段                          │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  5. QA 提示词构建                                                  │
│     构建包含上下文的提示词，含 [citation:N] 引用格式                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  6. 流式生成 + 返回                                                │
│     先下发 Base64 编码的引用上下文                                │
│     再逐块返回 LLM 生成文本（SSE）                                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  7. 结果保存                                                       │
│     完整回答写回 AI 消息 content 字段                              │
└─────────────────────────────────────────────────────────────────┘
```

## SSE 流式响应

### 响应格式

```javascript
// 单条 SSE 消息
data: {"text": "<base64_ctx>__LLM_RESPONSE__<文本块>"}

// 示例
data: {"text": "eyJjaXRhdGlvbnMiOiBbXX0=[citation:1]相关内容...__LLM_RESPONSE__这是"}
data: {"text": "[citation:2]另一段...__LLM_RESPONSE__根据文档，"}
data: {"text": "__LLM_RESPONSE__结论如下。"}
```

### 前端解析

1. 解析 `base64_ctx` 获取引用上下文
2. 按 `__LLM_RESPONSE__` 分隔符拆分
3. 拼接文本块形成完整回答

## 关键代码

### RagService

```python
async def generate_response(
    self,
    chat_id: int,
    user_message: str,
    settings: AiRuntimeSettings
) -> AsyncGenerator[str, None]:
    """流式 RAG 响应生成"""

    # 1. 保存消息
    user_msg = Message(role="user", content=user_message)
    assistant_msg = Message(role="assistant", content="")
    db.add_all([user_msg, assistant_msg])

    # 2. 加载知识库检索器
    with ai_runtime_scope(settings):
        embeddings = EmbeddingsFactory.create(settings)
        vector_store = ChromaVectorStore(embeddings)

        # 3. History-aware retrieval (LCEL)
        retrieval_chain = (
            {"chat_history": RunnableLambda(lambda x: get_chat_history(chat_id)),
             "question": RunnablePassthrough()}
            | rewrite_question_prompt
            | llm
            | StrOutputParser()
        )
        rewritten_question = await retrieval_chain.ainvoke(user_message)

        # 4. 检索相关文档
        docs = vector_store.similarity_search(rewritten_question, k=5)

        # 5. 构建引用上下文
        citation_context = json.dumps({"citations": [...]})

        # 6. 流式生成
        qa_prompt = build_qa_prompt(docs, user_message)
        async for chunk in llm.astream(qa_prompt):
            # 先发送引用上下文，再发送文本块
            yield f'{{"text": "{base64.b64encode(citation_context).decode()}_LLM_RESPONSE_{chunk}"}}'
```

### conversation_service.py

```python
async def create_chat(db: Session, user_id: int, name: str, kb_ids: list[int]) -> Chat:
    """创建对话并关联知识库"""
    chat = Chat(user_id=user_id, name=name)
    db.add(chat)
    db.flush()

    # 关联知识库
    chat.knowledge_bases = db.query(KnowledgeBase).filter(
        KnowledgeBase.id.in_(kb_ids)
    ).all()

    db.commit()
    return chat
```
