# 设计系统与样式约定

本文档与 [`tailwind.config.ts`](../tailwind.config.ts)、[`src/app/globals.css`](../src/app/globals.css) 为**唯一配色与排版来源**。新增页面与组件应优先使用**语义 token**，避免在业务代码中散落 `gray-*`、`blue-*` 等原子色（除非评估状态等明确需要独立语义色）。

## 1. 色彩 token（Tailwind `theme.extend.colors`）

| Token | 含义 | 典型用途 |
|--------|------|----------|
| `ink` | 主文字 | 标题、正文、强调标签 |
| `muted` | 次要文字 | 说明、占位、元信息 |
| `canvas` | 画布底色（变量） | 与 `--background` 配合 |
| `surface` | 卡片/面板背景 | 白底内容区、弹窗、侧栏内容 |
| `surface-muted` | 浅底区分区块 | 列表悬停、次级面板、上传区 |
| `border` | 描边与分割线 | 边框、`divide-*` |
| `accent` | 品牌主色（青绿） | 主按钮、链接、选中态、聚焦环 |
| `accent-hover` | 主色悬停 | `hover:bg-accent-hover` |
| `accent-muted` | 主色浅底 | 选中条、提示条、图标底 |

**在 `accent` 背景上的文字**请使用 `text-surface`（`surface` 为 `#ffffff`），以保证对比度；不要用 `text-white`，便于与 token 命名一致。

## 2. 字体

| Token | 用途 |
|--------|------|
| `font-display`（Fraunces） | 产品名、部分标题（如「工作台」、RAG 配置标题） |
| `font-sans` / 默认 body（IBM Plex Sans） | 正文、表单、列表 |

根布局见 [`src/app/layout.tsx`](../src/app/layout.tsx)。

## 3. 组件与页面约定

- **主操作按钮**：`bg-accent text-surface hover:bg-accent-hover`；禁用：`disabled:bg-border` 或 `disabled:opacity-50`。
- **次要按钮 / 幽灵按钮**：`border border-border bg-surface-muted text-ink hover:bg-surface-muted`。
- **输入框聚焦**：`focus:border-accent focus:ring-2 focus:ring-accent/30`（或与 `focus:ring-accent` 等价组合）。
- **破坏性操作**：仍使用 `red-*`（删除、错误 Toast），与品牌色区分。
- **评估 / 任务状态**：使用 [`EvaluationStatusBadge`](../src/components/EvaluationStatusBadge.tsx) 的琥珀、天青、翠、玫等语义色，**不**强行改成 `accent`。
- **对话页**：用户气泡 `bg-accent text-surface`；助手气泡 `bg-surface border-border`；引用链接 `text-accent`。

## 4. 背景与动效（globals.css）

- `body` 使用低对比多层径向渐变，与 `accent` 呼应。
- 列表入场：父级加 `stagger-reveal`，子项错开 `fade-in`。
- 滚动条：`.scrollbar-thin` 的滑块颜色与 `border`（`#e4e2dd`）对齐。
- **Markdown**（`.md-content`）：正文色为 `ink`；行内代码底 `#f1f0ed`（`surface-muted`）；链接与引用色为 `accent` 十六进制（与 token 一致）。

## 5. 维护与批量替换

- 新增批量迁移时可参考脚本 [`scripts/unify-design-tokens.mjs`](../scripts/unify-design-tokens.mjs)（`npm run unify-tokens`）。**合并前请 diff**，避免覆盖状态色等特殊类名。
- 与 **frontend-design** skill 对齐的原则：避免 Inter/Roboto + 默认蓝紫堆叠；本项目已固定为暖灰底 + 青绿主色，实现上通过 token 落地。
